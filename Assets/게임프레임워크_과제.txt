과제. 일반 멤버 변수/함수와 정적 멤버 변수/함수를 비교하여 설명하시오

일반 멤버 변수 -> 정적 변수와 달리, 객체를 생성해야 합니다 
                        객체를 생성하기 때문에 this를 사용할 수 있습니다
                        보통 사용할때부터 수명이 시작됩니다 (static보다 효율up)
                        이중 지역변수의 수명은 해당 함수 시작부터 함수 종료까지 이며, 
             함수 -> 마찬가지로, 외부에서 접근하기 위해 객체를 생성해야 하며,
                         this를 사용할 수 있습니다 
 
static 멤버 변수 --> 객체를 생성하지 않고 정의만 ( 헤더가 아닌 cpp에서 ) 한 후 외부에서 접근할 수 있습니다
                           객체를 생성하지 않기 때문에 this를 사용할 수 없습니다 
                           일반 멤버 변수와 달리, 수명이 프로그램 시작부터, 종료까지 지속됩니다 (프로그램 끝날때까지 값 유지)
정적 함수 --> static 함수는 객체를 생성하지 않고 이름만으로 호출할 수 있으며
                   this를 사용할 수 없습니다 
                   단, static 함수내 멤버 함수또한 정적 변수여야만 합니다 

정적 멤버 변수의 정의 및 초기화를 하지  않으면 에러가 나는 이유
--> 일반 멤버 변수는 선언 시, Stack메모리에 따로 저장이 되지마느
